# ReScript Project Template

The only official ReScript starter template.

## Installation

```sh
npm install
```

## Build

- Build ppx: `npm run build:ppx`
- Build: `npm run build`
- Clean: `npm run clean`
- Build & watch: `node src/FooBindBar.bs.js`
